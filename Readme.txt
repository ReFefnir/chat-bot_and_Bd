Запуск программы: 
Вводите имя пользователя
Вводите нужную строчку с двумя городами (сначала город-начало_пути, затем город-конец_пути).
(до, после и между ними допустимы любые символы)
Если двух городов нет в строке или же в базе данных (или городов в строке больше двух) - программа сообщит о том сообщение о неверном вводе.
Если они есть, программа выведет их названия в нужном порядке.

Для корректной работы программы необходима база данных mysql. Для работы, чисто теоретически, достаточно одной таблицы cities с полями "id - Int" и "name - String". Для тестирования остальных функций работы с бд нужны другие таблицы, описания которых даны в параметрах соответствующих функций.
