Запуск программы: 
Вводите имя пользователя
Для поиска городов:
Вводите нужную строчку с двумя городами (сначала город-начало_пути, затем город-конец_пути).
(до, после и между ними допустимы любые символы)
Если двух городов нет в строке или же в базе данных (или городов в строке больше двух) - программа сообщит о том сообщение о неверном вводе (с некоторой информацией).
Если они есть, программа выведет их названия в нужном порядке.
Для добавления города:
Вводите "/AddCity", а затем необходимые параметры (Name continent Country Latitude Longitude). Вместо пробелов в названии города используйте "_". При занесении в бд они будут заменены на пробелы.
Для корректной работы программы необходима база данных mysql. Для работы, чисто теоретически, достаточно одной таблицы cities с полями "id - Int", "Name - String", "Continent - String", "Country - String", "Latitude - Double", "Longitude - Double". Для тестирования остальных функций работы с бд нужны другие таблицы, описания которых даны в параметрах соответствующих функций.
